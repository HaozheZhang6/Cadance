这里有几个“稍简单点”但标准化程度极高的单体件：🟠 Phase 2.5: Intermediate (平衡视觉复杂性与脚本实现)Family建议标准核心几何特征价值说明wing_nut (蝶形螺母)DIN 315中央螺纹座 + 两个对称的弧形翅膀。典型的“非回转体”对称件。在脚本中只需画出半个翅膀的 Profile 镜像即可。star_knob (星形/五星把手)DIN 6336多边形星状外轮廓 + 圆润的倒角过渡。纯粹的 2D Profile 阵列拉伸。难点在于星角处的大圆角平滑过渡，在视觉上非常有识别度。grooved_pin (槽销)DIN 1473 / ISO 8740圆柱表面有三个等间距的、挤压出来的纵向凹槽。只需要在圆柱体上做三个简单的布尔切除。虽然简单，但这是标准件库中非常专业的定位零件。grease_nipple (黄油嘴/油嘴)DIN 71412 (H1型)六角基座 + 带球面的头部。纯单体回转体件。虽然有复杂的轮廓（球头、颈部），但在 CadQuery 中就是一个 revolve 操作。lifting_eye_nut (吊环螺母)DIN 582环形吊圈 + 底部的螺纹凸台。比起吊环螺栓，它的过渡段更短，结构更紧凑。只需要处理好环形扫掠与圆柱基底的融合。
