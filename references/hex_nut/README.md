# ISO 4032 — Hexagon Nuts, Style 1

## Standard
- **ISO 4032:2012** — Hexagon nuts, style 1 — Product grades A and B
- Equivalent: DIN 934 (superseded by ISO 4032)

## What It Covers
Standard (style 1) hexagonal nuts for general fastening with metric coarse threads.
Defines: nominal thread diameter M, across-flats s, across-corners e, and nut height m.

Thread form: ISO 68-1 (metric coarse, 60° angle)
Tolerance: Product grade A for M1.6–M16; grade B for M16–M39.

## Scope Parameters (mm)
- Nominal diameter: M3 – M48
- Nut height (m) is approximately 0.8 × M for small sizes, scaling to ~0.79 × M for larger sizes
