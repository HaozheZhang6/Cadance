# ISO 4014 — Hexagon Head Bolts (Partial Thread)

## Standard
- **ISO 4014:2011** — Hexagon head bolts — Product grades A and B (partial thread)
- Equivalent: DIN 931 (superseded by ISO 4014)
- Companion: ISO 4017 (full thread version)

## What It Covers
Hexagonal-head bolts with a partial threaded shank (unthreaded grip length on shank).
Defines: nominal diameter M, thread pitch (coarse), across-flats width s, head height k, and thread length b.

Thread form: ISO 68-1 (metric coarse, 60° angle)
Tolerance grade: Product grade A for M1.6–M16; grade B for M16–M64.

## Scope Parameters (mm)
- Nominal diameter: M3 – M48
- Standard lengths: ISO 888 preferred series — 8, 10, 12, 16, 20, 25, 30, 35, 40, 45, 50, 55, 60, 65, 70, 80, 90, 100, 110, 120, 130, 140, 150, 160, 180, 200, 220, 240, 260, 280, 300 mm
