# DIN 8187 / ISO 606 — Sprockets for Roller Chains (B-Series)

## Standards
- **ISO 606:2015** — Short-pitch transmission precision roller and bush chains, attachments and associated chain sprockets
- **DIN 8187** — Roller chains for power transmission; B-series (metric); superseded by ISO 606 but designation still in use
- Note: DIN 8187 = ISO 606 B-series (European/metric). The US A-series is ANSI B29.1.

## What It Covers
Defines the chain geometry (pitch, roller diameter, inner width, plate thickness) and sprocket tooth profile for B-series metric roller chains. Key sprocket dimensions are derived from the chain pitch and number of teeth.

### Key Formulas (from ISO 606 / sprocket geometry)
- **PCD** (pitch circle diameter) = p / sin(π / z)
- **Tip diameter** Da ≈ PCD + 0.8 × dr
- **Root diameter** Df = PCD − dr
- where p = chain pitch, z = tooth count, dr = roller diameter

## Scope Parameters
- Chain designations: 04B through 48B (pitch 6 mm to 76.2 mm)
- Tooth count z: practical range 9–120; commercial sprockets typically 9–80
- Disc thickness: 0.6 – 1.4 × dr (roller diameter)
- Hub optional (z ≥ 15 typical for hub variant)
- Keyway: DIN 6885 proportions by bore diameter
